Student Books API

Aναζητήστε εύκολα βιβλία Πληροφορικής

Τα API στην οικογένεια API των Βιβλίων Student Books σας επιτρέπουν να φέρετε λειτουργίες του Google Books στον ιστότοπο ή την εφαρμογή σας. Το νέο Student Books API σάς επιτρέπει να εκτελέσετε τις περισσότερες από τις λειτουργίες που μπορείτε να κάνετε αλληλεπιδραστικά στον ιστότοπο των Βιβλίων Google. Το ενσωματωμένο API προβολής σας επιτρέπει να ενσωματώσετε το περιεχόμενο απευθείας στον ιστότοπo. Στο Student Books API προσπαθούμε συνεχώς να παρέχουμε πληροφορίες σε όσο το δυνατόν περισσότερους ανθρώπους και τα API μας σχεδιάστηκαν με αυτό το στόχο. Το API δεν προορίζεται να χρησιμοποιηθεί ως αντικατάσταση των εμπορικών υπηρεσιών. Χρησιμοποιείται για την αναζήτηση συγκεκριμένων βιβλίων πληροφορικής και πιο ειδικά της php,java,javascript και visual basic.

Το Student Books API σας παρέχει πρόσβαση σε πολλές από τις λειτουργίες που είναι διαθέσιμες στον ιστότοπο των Βιβλίων Google.Μερικά από τα κύρια χαρακτηριστικά που παρέχει το API είναι: Αναζήτηση και περιήγηση στη λίστα των βιβλίων που αντιστοιχούν σε ένα δεδομένο ερώτημα. Προβολή πληροφοριών σχετικά με ένα βιβλίο, συμπεριλαμβανομένων των μεταδεδομένων, τη διαθεσιμότητα και την τιμή.

Ενσωματωμένο API προβολής

Το ενσωματωμένο API προβολής σας επιτρέπει να ενσωματώσετε περιεχόμενο βιβλίου από τα Βιβλία Google απευθείας στην ιστοσελίδα. Η προσθήκη λειτουργιών προεπισκόπησης βιβλίου στον ιστότοπό είναι σχετικά απλή. Mπορείτε να κάνετε μεγέθυνση ή σμίκρυνση, να επισημάνετε όρους αναζήτησης και ούτω καθεξής. Μπορείτε επίσης να λάβετε τον τρέχοντα αριθμό σελίδας,καθώς ο χρήστης περιηγείται στο βιβλίο.

Student Books API

Ο οδηγός αυτός φτιάχτηκε για να προσφέρει πληροφορίες για τη χρήση της ιστοσελίδας.Το student books API είναι ένα site το οποίο χρησιμοποιείται για να αναζητεί βιβλία και να φορτώνει δεδομένα που είναι αποθηκευμένα στο google Books API.Η αρχική σελίδα αποτελείται από ένα μενού με τις ιδιότητες home,guides,samples.Το home θα μεταβαίνει αυτόματα στην αρχική σελίδα μόλις ο χρήστης θα κάνει κλικ πάνω σε αυτό.Το guide θα περιέχει ένα μενού, το οποίο θα δίνει στον χρήστη οδηγίες για το πώς θα μπορέσει να χρησιμοποιήσει το site.Tο samples περιέχει ένα σετ βιβλίων και μας πληροφορεί για διάφορα χαρακτηριστικά που διαθέτει. Στην παρακάτω εικόνα φαίνεται η αρχική σελίδα, η οποία παρέχει τις πιο βασικές λειτουργίες.




Επιπλέον αποτελείται από ένα search box, το οποίο αναζητεί βιβλία που περιέχονται στο google API και μόλις ο χρήστης κάνει κλικ στο search, εμφανίζονται στο κέντρο της αρχικής σελίδας τα βιβλία που περιέχουν τον τίτλο, το όνομα του συγραφέα και επιπλέον πληροφορίες.
