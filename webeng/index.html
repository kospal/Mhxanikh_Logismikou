Ενσωμάτωση με το χώρο αποθήκευσης των Βιβλίων Google

Χρησιμοποιώντας το API Βιβλίων Goo- gle, η εφαρμογή σας μπορεί να πραγμα- τοποιεί αναζητήσεις πλήρους κειμένου και να ανακτά πληροφορίες βιβλίων, δυ- νατότητα προβολής και διαθεσιμότητα του ηλεκτρονικού βιβλίου. Το νέο API Google Books σας επιτρέπει να εκτελέσετε με προγραμματισμό τις περισσότερες από τις λειτουργίες που μπορείτε να κάνετε.
